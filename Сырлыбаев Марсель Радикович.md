# Проект "Агентство Спектр"

## Назначение
WPF-приложение для автоматизации работы строительной организации.

## Основные функции
- **Клиенты** – добавление, редактирование, удаление, просмотр
- **Товары** – учёт стройматериалов (название, категория, цена, остаток)
- **Сотрудники** – учёт персонала (ФИО, должность, телефон, зарплата)
- **Заказы** – создание заказов, изменение статуса (новый/завершён/отменён)
- **Отчёты** – выручка, количество заказов, популярные товары

## Технологии
- Язык: C#
- Фреймворк: WPF (.NET Framework 4.7.2)
- База данных: Microsoft SQL Server (база OPT)
- Доступ к данным: Dapper + ADO.NET Entity Data Model
- Система контроля версий: Git, GitHub
